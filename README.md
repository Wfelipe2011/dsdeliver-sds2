#Ds_Deliver

# Sobre o Projeto

https://dsdeliver.netlify.app/

Ds_Deliver é uma aplicação full stack construida na 2° edição da Semana DevSuperior.

A aplicação simula um serviço de delivery, onde o cliente faz o pedido.
